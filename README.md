# React SPA (Vite)

This project is a Single Page Application (SPA) built with React and Vite.  
It uses a clean and simple folder structure for components, pages, and assets.

## Project Structure

.
├── public/
├── src/
│   ├── assets/
│   │   └── images/
│   ├── components/
│   ├── pages/
│   ├── App.jsx
│   ├── main.jsx
│   └── index.css
├── index.html
├── package.json
├── package-lock.json
├── vite.config.js
├── eslint.config.js
└── README.md

## Technologies Used

- React
- Vite
- JavaScript (ES6+)
- CSS
- ESLint

## Scripts

Install dependencies:
npm install

Start development server:
npm run dev

Build for production:
npm run build

Preview production build:
npm run preview

## Folder Overview

src/components  
Reusable UI components

src/pages  
Page-level components used for routing and layout

src/assets/images  
Static images and media files

App.jsx  
Main application component

main.jsx  
Application entry point

## Purpose

This project is a learning and practice SPA focused on:
- Component-based architecture
- Clean project structure
- Modern frontend tooling with Vite
